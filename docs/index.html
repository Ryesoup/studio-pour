브랜드 철학

Studio Pour는 익숙한 일상에 따뜻함과 감성을 ‘따르듯’ 담아냅니다.
단순한 기능을 넘어, 루틴을 정돈해주는 디자인을 만듭니다.

비전

감각적인 일상

당신의 루틴에 어울리는 디자인을 고민합니다.

지속 가능한 제작

국내 생산과 고품질 소재를 바탕으로, 오래도록 곁에 두는 제품을 만듭니다.

사용자 중심

실제 사용자의 루틴과 피드백을 반영해 제품을 발전시킵니다.
